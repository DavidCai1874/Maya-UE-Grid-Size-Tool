# Maya-UE-Grid-Size-Tool


Note: This tool still needs further polish and documentation.

This tool is for a smoother workflow between Maya and UE. It sets the grid size to match UE's, and with simple scale hide/show animation (cause now there's only a few small objects). And with a smart version save to auto save a new version based on current path
